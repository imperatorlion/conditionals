# Conditional Quest

Open `index.html` in a browser to play.

## Browserlink (lokale server)

1. Open een terminal in deze map.
2. Run:
   ```sh
   python -m http.server 8000
   ```
3. Open in je browser: **http://localhost:8000/**

## Besturing

- Bewegen: pijltjestoetsen
- Vraag openen: tegen schatkist lopen
- Minigame: bewegen met pijltjes, schieten met **Enter**
